Конструктор и деструктор. Перегрузка конструктора

  • Конструктор — вызывается при создании объекта. Имя совпадает с именем класса. Может быть перегружен (разные наборы параметров).
  • Деструктор — ~ClassName(), вызывается при уничтожении объекта, освобождает ресурсы. Не перегружается.
  • Если конструктор не определён, компилятор создаёт конструктор по умолчанию (без параметров).
